ROS 2 Engineering Skills

AI 코딩 에이전트(Claude Code, Cursor, Gemini CLI)에 프로덕션급 ROS 2 개발 전문성을 제공하는 에이전트 스킬 모듈.

GitHub

소개

Progressive disclosure 아키텍처를 채용하여 SKILL.md의 간결한 라우팅과 20개 참조 파일의 상세 패턴으로 구성됩니다. Humble, Jazzy, Kilted, Rolling 등 다양한 ROS 2 배포판을 지원하며, DDS 설정부터 플릿 배포까지 전 범위를 커버합니다.

주요 기능

  • 언어 선택(rclcpp vs rclpy), QoS 프로파일, 노드 라이프사이클 패턴에 대한 의사결정 프레임워크
  • DDS 설정, 실시간 커널 튜닝, ros2_control 하드웨어 인터페이스
  • Nav2 내비게이션, MoveIt 2 매니퓰레이션, micro-ROS 임베디드 시스템
  • 보안(SROS2), 다중 로봇 플릿 관리, Docker 배포 및 크로스 컴파일
  • 프로덕션 실패 모드를 강조하는 안티패턴 문서
  • 137개 테스트 케이스, 99% 코드 커버리지

기술 스택

언어Python 3.10-3.12, Shell
테스트pytest, Hypothesis, flake8, mypy
CI/CDGitHub Actions (4 워크플로우)
