다중 로봇 행성 탐사 시뮬레이션

다중 로봇 자율 협력 시스템을 활용한 외계 행성 탐사 시뮬레이션.

GitHub Demo
기간 2024.12.17 ~ 2024.12.23
인원 3명

담당 역할

ROS2 통신 각 노드 개발 관제시스템 map/odom Nav2 파라미터 설정

소개

외계 행성 탐사 시뮬레이션입니다. 로봇1(지형 탐사/정찰, 외계인 발견 시 경고), 로봇2(로봇팔 지질탐사, 경고 시 복귀), 로봇3(외계인, 추격 → 홈 복귀 시 초기 위치 복귀). dolly + manipulator 제작 시도 → spawn 실패 → TurtleBot3 waffle 개조하여 외계인 부착 방식으로 해결했습니다.

주요 기능

  • 달 표면 기반 행성 탐사 시나리오
  • 외계인 침입자 배치 및 자율 이동 시뮬레이션
  • 침입자 감지 시 로봇 자동 귀환
  • 2차 로봇의 매니퓰레이터를 이용한 표면 시료 수집
  • 우주 정거장 제어 시스템 — 실시간 로봇 위치, 침입자 위치 맵핑
  • 미확인 생명체 발견 시 알림 팝업 및 로깅
  • PyQt5 기반 미션 컨트롤 GUI
