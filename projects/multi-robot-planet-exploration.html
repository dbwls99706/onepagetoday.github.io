다중 로봇 행성 탐사 시뮬레이션

다중 로봇 자율 협력 시스템을 활용한 외계 행성 탐사 시뮬레이션.

GitHub

소개

달 표면을 기반으로 한 외계 행성 탐사 시나리오입니다. 외계인 캐릭터를 침입자로 배치하고, 로봇 감지 시 자율적으로 이동합니다. 다중 로봇이 협력하여 탐사, 침입자 감지, 표면 시료 수집을 수행합니다.

주요 기능

  • 달 표면 기반 행성 탐사 시나리오
  • 외계인 침입자 배치 및 자율 이동 시뮬레이션
  • 침입자 감지 시 로봇 자동 귀환
  • 2차 로봇의 매니퓰레이터를 이용한 표면 시료 수집
  • 우주 정거장 제어 시스템 — 실시간 로봇 위치, 침입자 위치 맵핑
  • 미확인 생명체 발견 시 알림 팝업 및 로깅
  • PyQt5 기반 미션 컨트롤 GUI

기술 스택

언어Python 3
OSUbuntu 22.04
프레임워크ROS 2 Humble
시뮬레이션Gazebo 11, RViz2
네비게이션Nav2, TurtleBot3
GUIPyQt5
