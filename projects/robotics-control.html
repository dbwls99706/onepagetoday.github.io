Robotics Control

다자유도 로봇 시스템의 동역학 모델링, PID 제어, 파라미터 추정을 MATLAB으로 구현한 5개 과제 모음.

GitHub

프로젝트 목록

  • Project 1: 3-DOF 자유 낙하 시뮬레이션 — 라그랑주 기반 동역학 모델링, ODE45 적분, 관절 각도 변화 시각화
  • Project 2: 2-DOF 파라미터 추정 — 칼만 필터링 및 오차 최소화 기법, 수렴 속도 비교
  • Project 3: 1-DOF 조인트 공간 PID 제어 — 단일 링크 PID 컨트롤러 설계, 스텝 응답 분석
  • Project 4: 2-DOF 직교 공간 PID 제어 — 원형 경로 엔드이펙터 궤적 추적, 야코비안/관성/코리올리/중력 보상
  • Project 5: 3-DOF 확장 — Project 3을 3-DOF 조인트 공간으로, Project 4를 3-DOF 직교 공간으로 확장 (역기구학 포함)
