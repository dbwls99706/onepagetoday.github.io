YOLOv8 on Raspberry Pi 4

라즈베리파이 4B에서 YOLOv8 객체 인식을 실행하는 ROS 2 패키지. 로봇 기반 전동 휠체어 개조 시스템의 자율 주행을 위한 실시간 객체 감지.

GitHub Demo
기간 2024.01 ~ 2024.06
인원 4명
유형 캡스톤 설계 프로젝트

담당 역할

ROS2 통신 YOLO 장애물 감지 소프트웨어 경량화 하드웨어 구성

프로젝트 목표

기존 수동 휠체어에 전동 기능을 추가하여 사용자의 이동을 지원하는 시스템을 개발했습니다. 저비용의 소재와 간단한 설계로 전동 휠체어보다 경제적인 해결책을 제공하면서, 기존 수동 휠체어에 부착 가능한 모듈형 로봇을 설계하는 것이 핵심이었습니다.

주요 기능

  • 부착형 모듈 — 일반 휠체어에 모듈을 부착하여 수동식에서 조이스틱 조종 가능한 전동식으로 전환
  • 장애물 감지 — 카메라 바운딩 박스를 활용해 2~3m 안의 장애물 인식, GPIO 23/24번에 True 신호 출력하여 STM32에 전달
  • 소프트웨어 경량화 — 라즈베리파이에서 YOLO 동작을 위해 최소한의 필요 기능만 남김 (yolov8n.pt)
  • 균형 조정 — MPU6050 센서 값을 기반으로 리니어 모터 조정, 좌석 수평 유지
  • ROS 2 Launch 설정을 통한 자동화된 노드 시작
