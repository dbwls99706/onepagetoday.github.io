Tello 드론 컬러 제어

MATLAB으로 Ryze Tello 드론을 제어하여 빨강 → 초록 → 파랑 순서로 색상을 인식하고 비행 명령을 실행.

GitHub

소개

MATLAB과 Ryze Tello 드론을 이용해 카메라로 빨간색 → 초록색 → 파란색 순서로 물체를 인식하고, 각 색상에 따라 다른 비행 동작을 수행합니다.

비행 로직

  • 지정된 고도에서 이륙
  • 오른쪽으로 0.2m 단위 이동
  • 빨간색 감지 시 — 0.2m 단위 상승
  • 초록색 감지 시 — 오른쪽 이동 계속
  • 파란색 감지 시 — 착륙
  • 무한 루프 실행, CTRL+C로 수동 종료
