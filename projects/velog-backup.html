Velog Backup

Velog 블로그 포스트를 이미지 포함하여 안전하게 백업하는 무료 웹 서비스.

GitHub

소개

Velog 포스트를 이미지를 포함하여 완전히 백업하고, GitHub 저장소와 동기화하며, 이메일 알림까지 지원하는 무료 서비스입니다. Velog 포스트가 삭제되어도 백업본이 영구 보존됩니다.

주요 기능

  • 이미지 포함 백업 — 포스트 이미지를 실제 파일로 다운로드
  • 체계적 폴더 구조 — 제목별 폴더로 정리된 ZIP 다운로드
  • GitHub 저장소 동기화 — 비공개 GitHub 저장소에 자동 커밋
  • 이메일 알림 — 백업 완료/실패 시 알림
  • 서버 저장 — 데이터베이스에 안전하게 보관
  • 영구 보존 — Velog 포스트 삭제 후에도 백업 유지
  • 완전 무료

기술 스택

프론트엔드Next.js 14, React 18, Tailwind CSS, TypeScript
백엔드FastAPI, SQLAlchemy, Python 3.11
데이터베이스PostgreSQL (Supabase)
인증GitHub OAuth, JWT
배포Vercel (프론트) / Railway (백엔드)
