카페 서빙 로봇

오피스 카페 커피 배달 서비스를 위한 주문 시스템과 자율 서빙 로봇 시뮬레이션.

GitHub

소개

Gazebo와 RViz2 시뮬레이션 환경에서 오피스 카페의 주문부터 배달까지 전 과정을 자동화합니다. 테이블 주문 시스템, 주방 디스플레이, 자율 서빙 로봇이 연동됩니다.

주요 기능

  • 테이블 주문 시스템 — 다양한 메뉴 옵션, 사무실 배달/외부 픽업 구분
  • 주방 디스플레이 — 주문 교차 확인, 취소 사유 제공, 완성된 주문을 서빙 로봇에 전달
  • 자율 서빙 로봇 — 다중 테이블 배달, 퀴즈 및 쿠폰 제공
  • 데이터베이스 저장 — 전체 거래 기록
  • 매출 분석 — Top 3 메뉴 통계
  • 시스템 로깅 — Info, Warning, Error 레벨
