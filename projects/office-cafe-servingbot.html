카페 서빙 로봇

오피스 카페 커피 배달 서비스를 위한 주문 시스템과 자율 서빙 로봇 시뮬레이션.

GitHub Demo
기간 2024.11.12 ~ 2024.11.18
인원 4명
유형 서비스 로봇 및 관제 시스템

담당 역할

Kitchen GUI 구성 ROS2 통신 총괄 SQLite DB 저장/활용

소개

사내 카페와 서빙 로봇을 결합한 커피 배달 서비스입니다. 실제 카페와 유사한 환경을 구현하고, 배달/픽업 주문 세분화, 주문 취소 사유 제공, 서빙 로봇의 친근감 있는 UX를 목표로 했습니다.

주요 기능

  • 테이블 오더 — 다양한 메뉴 및 커스텀 옵션, 겨울 시즌 메뉴(군밤고구마 라떼, 군고구마, 붕어빵), 배달과 픽업 세분화
  • 주방 디스플레이 — 테이블 오더와 주문 정보 비교하여 오조리 방지, 주문 취소 사유 표시, Top3 매출 분석
  • 서빙 로봇 — 눈 모양으로 친근감 제공, 친절한 메시지와 퀴즈로 특별한 경험, 다른 테이블 배달 기능
  • 데이터베이스 — pretty table을 이용해 SQLite에 저장된 데이터 표시
  • 시스템 로깅 — Info, Warning, Error 레벨
