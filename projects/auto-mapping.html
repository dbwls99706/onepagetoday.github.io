자율 매핑 & 장애물 회피

TurtleBot4 기반 자율 환경 매핑, 장애물 회피, 실시간 객체 추적 시스템.

기간 2024.11.26 ~ 2024.12.02
인원 3명
유형 SLAM 기반 로봇청소기

담당 역할

자동 맵핑 코드 구현 wall_follow 로직 구현

소개

OccupancyGrid 데이터를 분석하여 미탐색 영역, 자유 공간, 벽을 구분하고 자율적으로 미매핑 영역을 탐색하여 환경 지도를 완성합니다. 매핑 완료 후 벽면 추적(Wall Following) 행동으로 전환하며, ORB 알고리즘으로 실시간 객체를 추적합니다.

주요 기능

  • 자동 매핑 — 미탐사 영역(unknown_cells) 탐지, DBSCAN 클러스터링으로 프론티어 그룹화, Nav2/SLAM 튜닝
  • 벽면 추적 — 매핑 완료 후 /exploration_done 토픽 수신, 10초 간격 방향 전환
  • 객체 추적 — ORB 알고리즘 기반 실시간 감지, 해밍 거리 기반 특징점 정렬
  • 파라미터 최적화를 통한 환경 지도 완성
