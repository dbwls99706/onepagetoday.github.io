OpenSource Contribution Card

GitHub 프로필 README에 외부 오픈소스 기여를 자동으로 표시하는 SVG 위젯.

GitHub

소개

본인이 소유하지 않은 프로젝트에 머지된 PR만 필터링하여, 진정한 외부 오픈소스 기여만을 GitHub 프로필에 자동으로 표시합니다. GitHub Actions를 통해 하루 3회 자동 업데이트되며, 5가지 테마를 지원합니다.

주요 기능

  • 외부 기여만 필터 — 개인 저장소 제외, 머지된 PR만 표시
  • 자동 일일 업데이트 — GitHub Actions로 하루 3회 갱신
  • 5가지 테마 — light, dark, nord, dracula, tokyo night
  • 자동 테마 감지 — GitHub의 라이트/다크 모드에 맞춤
  • OSS Score 산출 — PR 수와 저장소 다양성 기반 기여도 점수
  • 날짜 범위, 조직, 기여 유형별 PR 필터링
  • Featured PR 수동 큐레이션
