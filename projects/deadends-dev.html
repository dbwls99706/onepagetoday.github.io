deadends.dev

AI 코딩 에이전트가 실패한 접근법을 피하고 작동하는 해결책을 찾도록 돕는 구조화된 실패 지식 인프라.

GitHub

소개

51개 기술 도메인에 걸쳐 2,089개의 에러 항목을 카탈로그화합니다. Python 88개, Node.js 70개, Docker 65개 등 포괄적인 커버리지를 제공하며, 각 에러에 대해 "시도하지 마세요(Dead Ends)"와 "이것을 시도하세요(Workarounds)"를 실패율/성공률과 함께 문서화합니다.

주요 기능

  • Dead Ends — 실패율이 문서화된 "시도하지 마세요" 접근법
  • Workarounds — 성공률이 포함된 "이것을 시도하세요" 해결책
  • 에러 전이 그래프 — 한 에러 다음에 어떤 에러가 오는지 추적
  • 8개 MCP 도구로 AI 에이전트 직접 통합
  • 18가지 디스커버리 포맷 — JSON API, llms.txt, JSON-LD, OpenAPI
  • Python SDK, CLI, MCP Server, RESTful API 제공

기술 스택

언어Python 3.10+
배포PyPI, Vercel
표준OpenAPI 3.1, Schema.org, llmstxt.org
