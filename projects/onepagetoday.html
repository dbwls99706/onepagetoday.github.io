One Page Today

아무것도 쓰지 않아도, 스마트폰 사용 흔적만으로 하루가 한 페이지 일기로 남습니다.

Google Play

소개

One Page Today는 사용자가 직접 글을 쓰지 않아도 스마트폰 사용 패턴을 분석하여 자동으로 하루를 기록해주는 Android 앱입니다. 메시지 빈도, 사진 촬영 시점, 이동 패턴, 앱 사용 흐름 등 이미 남긴 흔적이 하루를 말해줍니다.

주요 기능

  • 쓰지 않아도 되는 일기 — 메시지 빈도, 사진 촬영 시점, 이동 패턴, 앱 사용 흐름으로 하루를 자동 기록
  • 담백한 한 페이지 — 감정을 해석하지 않고 시간, 변화, 리듬만을 담백하게 정리
  • 위로하지 않고, 조언하지 않음 — 공유를 전제로 하지 않는, 오직 나만을 위한 조용한 기록
  • 하루의 끝, 최소한의 뿌듯함 — 오늘을 그냥 흘려보내지 않았다는 감각
