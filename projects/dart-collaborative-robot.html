DART 협동로봇 운영

두산로보틱스 DART 플랫폼을 활용한 협동로봇 동작 운영 실습. Pattern Pallet 및 안전 기능 최적화.

기간 2024.12.10 ~ 2024.12.16
인원 3명
플랫폼 두산로보틱스 DART

담당 역할

로직 구성 아이디어 제시 기반 환경 구성 좌표 고안

과제 내용

  • 과제1 — Pattern Pallet 실습: get_pattern_point() 응용. 빈 pallet에 전체 point 순차적 채워넣기 → 역순 가져오기. 프로그램 정지 후 중단 시점부터 재개, 임의의 특정 point부터 시작 가능
  • 과제2 — 안전 기능 최적화: 워크스페이스 외 공간 제한 구역 설정(TCP가 테이블 바닥 이하/밖으로 넘어가지 않게), 충돌 민감도 90%+, TCP 속도 50mm/s 이하, 손끼임 방지용 RS1 안전 정지 모드, 전체 작업 속도 200mm/s
  • 과제3 — 협동로봇 시스템 기획: 안전 기능과 안전장치를 활용한 협동로봇 시스템 구성

기술 스택
