VisionAMR Crime Tracker

YOLOv8 + Jetson Nano 기반 범죄 차량 감지 및 자율 추적 AI 감시 시스템.

GitHub

소개

CCTV 기반 경계 감시와 침입 감지, 범죄 차량과 일반 객체를 구분하는 분류 시스템, 범죄 차량 식별 시 자동 추적하는 자율 로봇을 결합한 AI 감시 시스템입니다.

주요 기능

  • CCTV 경계 감시 — 침입 감지 시 즉각 알림
  • 범죄 차량 분류 — 일반 객체와 범죄 차량 구분
  • 자율 추적 — 범죄 차량 식별 시 자동 추적, 장애물 회피
  • 관리 대시보드 — 보안 로그인, 실시간 CCTV/AMR 카메라 피드
  • SQLite3 데이터 관리 — 포렌식 분석 및 패턴 추적

기술 스택

언어Python 3, HTML, CSS, JavaScript
OSUbuntu 22.04 LTS
프레임워크ROS 2 Humble, Flask
시뮬레이션Gazebo 11, RViz2
비전OpenCV, YOLOv8, LabelImg
