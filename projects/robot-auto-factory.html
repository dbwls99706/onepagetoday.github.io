물류 자동화 시스템

TurtleBot3 + OpenManipulator-X 기반 컨베이어 벨트 부품 분류 및 운반 자동화 시스템.

GitHub Demo
기간 2024.11.19 ~ 2024.11.25
인원 3명
유형 Fulfilment 가상 서비스 환경 실습

담당 역할

ROS2 통신 구성 PyQT 커맨드센터 ArUco 마커 측정 Unity6 Digital Twin

소개

3개 과제로 구성된 프로젝트입니다: (1) OpenManipulator를 이용한 물류 공장(박스 판단 → 집어서 컨베이어 벨트로 이동), (2) ArUco 마커 활용 거리/각도 측정, (3) Unity6 기반 Digital Twin 환경 구성. 로봇 위치 분산 약 ±1cm에서도 안정적으로 동작합니다.

주요 기능

  • YOLO 기반 박스 분류 — Unity 6로 학습 데이터 생성, YOLOv8으로 실시간 인식
  • 관리 인터페이스 — 실시간 카메라 피드, YOLO 감지 결과 표시, 수동 컨베이어 제어
  • ArUco 마커 추적 — 글로벌 카메라 기반 마커 상대 위치/각도 측정
  • 로봇 위치 분산 ±1cm에서도 안정적 동작
  • 작업 완료 시 경과 시간 표시 알림

기술 스택

프레임워크ROS 2 Humble
하드웨어TurtleBot3 Waffle, OpenManipulator-X, Arduino, 컨베이어
