물류 자동화 시스템

TurtleBot3 + OpenManipulator-X 기반 컨베이어 벨트 부품 분류 및 운반 자동화 시스템.

GitHub

소개

ROS 2 기반 물류 자동화 시스템으로, TurtleBot3 Waffle과 OpenManipulator-X를 결합하여 컨베이어 벨트에서 부품을 분류하고 운반합니다. YOLOv8 인식 결과를 바탕으로 박스 분류 및 컨베이어 배치를 수행하며, 로봇 위치 분산 약 ±1cm에서도 안정적으로 동작합니다.

주요 기능

  • YOLO 기반 박스 분류 — Unity 6로 학습 데이터 생성, YOLOv8으로 실시간 인식
  • 관리 인터페이스 — 실시간 카메라 피드, YOLO 감지 결과 표시, 수동 컨베이어 제어
  • ArUco 마커 추적 — 글로벌 카메라 기반 마커 상대 위치/각도 측정
  • 로봇 위치 분산 ±1cm에서도 안정적 동작
  • 작업 완료 시 경과 시간 표시 알림

기술 스택

언어Python 3
OSUbuntu 22.04 LTS
프레임워크ROS 2 Humble
하드웨어TurtleBot3 Waffle, OpenManipulator-X, Arduino, 컨베이어
비전OpenCV 4.5.4, YOLOv8
GUIPyQt5
