파이프스풀 제작 자동화

기존 Xenomai RTOS + EtherCAT 제어 시스템에 CSV 궤적 재생 기능을 추가하고, Keyence LJX8 레이저 변위계 기반 보정 루프로 용접 갭(gap_x)을 4.3mm에서 0.4mm까지 줄였다.

기간 2025.06 ~ 진행 중
소속 엔바이어스 연구원
유형 파이프스풀 용접 자동화

담당 역할

CSV 궤적 재생 기능 추가 Keyence LJX8 레이저 보정 연속 보정 루프 확장

소개

파이프스풀(파이프 조립체)의 용접 공정을 자동화하는 시스템이다. 기존에는 사내에서 구축한 Xenomai RTOS + EtherCAT 기반 실시간 제어 스택이 동작하고 있었고, 이 위에 사전 계획된 궤적을 재생·보정하는 응용 계층을 추가했다.

오프라인에서 산출한 궤적을 CSV로 받아 재생하면서, 파이프 단면의 실측 갭을 Keyence LJX8 라인 레이저로 측정해 실시간 보정한다. 한 점 보정에서 시작해 연속 보정 루프로 확장하는 중이다.

주요 작업

  • 기존 실시간 제어 시스템 위에 CSV 궤적 파서·재생기 추가
  • Keyence LJX8 라인 레이저 변위계 연동, 용접 갭(gap_x) 측정 파이프라인 구성
  • 측정값 기반 보정 루프를 결합하여 gap_x 4.3mm → 0.4mm 달성
  • 단발 보정에서 연속 보정으로 확장하는 작업 진행 중

성과

레이저 변위계 기반 실시간 보정 도입으로 용접 갭 gap_x 4.3mm → 0.4mm(약 10배 정밀도 향상) 달성
